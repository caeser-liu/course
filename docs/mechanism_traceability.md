# 晶圆缺陷与工艺机理溯源说明

本说明对应 `goal.md` 中“结合机械与工艺知识的机理溯源分析”部分。模型训练完成后，可将预测类别、置信度和 Grad-CAM 热力图作为视觉证据，用于辅助判断缺陷可能来源。

## 类别与可能工艺来源

| 类别 | 空间特征 | 可能原因 | 关联工艺/设备 |
| --- | --- | --- | --- |
| Center | 缺陷集中在晶圆中心 | 中心温度、沉积速率或旋涂厚度异常 | CVD/PVD、涂胶、热处理 |
| Donut | 中心附近形成环状缺陷 | 光刻对准、旋涂流场或局部沉积异常 | 光刻、涂胶显影、薄膜沉积 |
| Edge-Loc | 局部边缘区域异常 | 边缘夹持、搬运接触或边缘局部刻蚀异常 | 搬运机械手、刻蚀、边缘清洗 |
| Edge-Ring | 边缘整圈或大范围环状异常 | 腔体边缘气流、温度场或等离子体分布不均 | 刻蚀、CVD、PVD |
| Loc | 局部点状或块状异常 | 颗粒污染、局部污染源、设备局部状态漂移 | 清洗、搬运、环境颗粒控制 |
| Random | 随机散布缺陷 | 随机颗粒污染、测试噪声或来料波动 | 洁净室、检测设备、前序工艺 |
| Scratch | 线状划痕 | 抛光垫颗粒、机械接触、搬运摩擦 | CMP、机械手、晶圆盒 |
| Near-full | 大面积接近满片异常 | 严重设备失效、整片工艺条件失控 | 沉积、刻蚀、热处理 |
| none | 无明显缺陷模式 | 正常样本或未出现典型空间缺陷 | 正常工艺窗口 |

## 使用 Grad-CAM 的判读方法

1. 对测试样本运行模型预测，记录类别和置信度。
2. 对同一图片生成 Grad-CAM 热力图。
3. 检查热力图高响应区域是否落在该类别的典型空间区域：
   - `Scratch` 应沿线状缺陷区域响应。
   - `Edge-Ring` 应在晶圆边缘环带响应。
   - `Center` 应在中心区域响应。
   - `Loc` 应在局部异常块附近响应。
4. 若热力图关注背景或无关区域，应降低该预测对工艺溯源的可信度，并检查数据预处理、类别不平衡和模型过拟合问题。

## 课程报告建议结构

1. 项目背景：半导体良率检测、晶圆缺陷空间模式、可解释 AI 的必要性。
2. 数据说明：WM-811K 数据来源、9 类标签、长尾分布和下采样策略。
3. 方法流程：数据预处理、ResNet-34 迁移学习、类别权重、评估指标。
4. 可解释分析：展示每类代表样本的原图、预测结果和 Grad-CAM 热力图。
5. 工艺溯源：结合上表说明模型识别结果与机械/工艺故障的可能对应关系。
6. 局限性：公开数据缺少真实设备日志，溯源只能作为工程假设，不能替代现场 SPC/FDC 数据验证。
